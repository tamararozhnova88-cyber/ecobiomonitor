# EcoBioMonitor — мониторинг биоразнообразия через IT-решения

## Описание проекта
EcoBioMonitor — учебный проект, представляющий собой прототип IT-системы
мониторинга биоразнообразия.  
Проект демонстрирует, как методы анализа данных и машинного обучения могут
использоваться для оценки состояния экосистем и выявления потенциальных
экологических угроз.

## Актуальность
Современные экосистемы испытывают возрастающее антропогенное и климатическое
воздействие. Традиционные методы экологического мониторинга требуют больших
затрат времени и ресурсов, а также подвержены человеческим ошибкам.
Использование IT-решений позволяет автоматизировать сбор, анализ и интерпретацию
данных о биоразнообразии.

## Используемые данные
- Открытые данные о биоразнообразии национальных парков (CSV)
- Демонстрация получения актуальных наблюдений через API:
  - iNaturalist
  - GBIF (Global Biodiversity Information Facility)

## Функциональные возможности
- анализ наблюдений видов по территориям;
- расчёт индексов биоразнообразия (видовое богатство, индекс Шеннона, индекс Симпсона);
- выявление аномалий с помощью алгоритма Isolation Forest;
- rule-based оповещения о потенциально проблемных территориях;
- визуализация ключевых показателей.

## Используемые технологии
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- REST API (iNaturalist, GBIF)

## Результаты
- проведён анализ данных биоразнообразия;
- рассчитаны ключевые экологические индексы;

- выявлены аномальные территории с потенциальными экологическими рисками;
- продемонстрирована возможность масштабирования системы за счёт внешних API.

## Выводы
Проект показывает, что IT-решения и методы машинного обучения могут быть
эффективным инструментом для мониторинга биоразнообразия и поддержки принятия
экологических решений.
